# Bus Seat Planner

Конструктор и просмотрщик схем мест в автобусе. Изначально проект был просто
скриптом для выбора мест в конкретной модели автобуса (Mercedes Sprinter),
теперь это универсальный инструмент: можно нарисовать схему салона любого
автобуса, настроить нумерацию и расположение мест, дверей, места гида/водителя,
а затем показывать гостям интерактивную схему с отметкой свободных/занятых мест.

## Возможности

- **Конструктор профилей** (`admin.html`) — редактор в виде таблицы: кликами
  отмечаете, где сиденья, где проход, где дверь, где место гида/водителя.
  По каждому профилю задаются:
  - количество столбцов/рядов;
  - схема автонумерации мест (например, "по колонкам, от носа к корме,
    внутри колонки сверху вниз");
  - положение двери (с какой стороны и в каком месте салона);
  - положение задних колёс (между какими колоннами мест);
  - нужно ли резервировать место для гида (учитывается в общей нумерации,
    но по умолчанию считается всегда занятым).
- **Автогенерация схемы автобуса** (`seatProfiles.js`) — по сохранённому
  профилю рисуется реалистичная SVG-схема: кузов, окна, дверь, руль,
  колёса, кресла со спинками и номерами. Схема включает встроенную легенду
  ("свободное место" / "занятое место") и полностью самодостаточна для
  экспорта в PNG (копирование в буфер / скачивание).
- **Несколько профилей на выбор** — можно создать сколько угодно схем
  автобусов, переключаться между ними, последний выбранный профиль
  запоминается в настройках браузера (localStorage).
- **Классический профиль** — исходная схема Mercedes Sprinter (20 мест)
  оставлена как встроенный профиль по умолчанию.
- Отметка свободных/занятых мест прямо на схеме (двойной клик по креслу),
  синхронизация со списком мест и со ссылкой (хэш в адресной строке),
  копирование итоговой картинки одним кликом.

## Как пользоваться

1. Откройте `avtobusJZ.html` — здесь отображается схема автобуса, отмечаются
   свободные/занятые места, копируется/скачивается итоговая картинка.
2. Откройте `admin.html`, чтобы создать свой профиль автобуса или
   отредактировать существующий. Сохранённые профили сразу становятся
   доступны для выбора в `avtobusJZ.html`.

Можете свободно копировать себе на сайт и использовать в любых целях.
